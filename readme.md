The current project is to make a recipe webpage. In the process of completing it, 
I learned quite a few things. It let me use the skills of formatting text using HTML, 
inserting links on a page and adjusting the sizes of images, as well as 
proper etiquette to use when linking to offsite webpages with HTML links. 
I also, in doing this, learned how to push to Git, how to create repositories,
and how to publish a webpage hosted on Git itself. 
All in all, this project wasn't too technically difficult for a beginner like me,
which is good, because it didn't make me feel out of my depth, but I could
also feel how I was using all the things I had previously been taught for this process.
I'm looking forward to the next project, and I'm going to remember that my repos actually exist
instead of doing one big commit at the end next time.
